# 后台管理系统-首页线框图

```
┌──────────────────────────────────────────────┐
│ 顶部导航栏：Logo | 系统名称 | 用户头像 | 退出 │
├──────────────────────────────────────────────┤
│ 侧边栏菜单：                                  │
│ [仪表盘] [用户管理] [题库管理] [数据统计]     │
│ [权限管理] [系统设置]                         │
├──────────────────────────────────────────────┤
│ 主内容区                                      │
│ ┌──────────────────────────────────────────┐ │
│ │ 统计卡片：                                │ │
│ │ ┌────────┐ ┌────────┐ ┌────────┐        │ │
│ │ │ 用户数 │ │ 题库数 │ │ 做题量 │        │ │
│ │ └────────┘ └────────┘ └────────┘        │ │
│ │                                          │ │
│ │ 快捷入口：                                │ │
│ │ [导入题库] [导出题库] [新增公告]         │ │
│ │                                          │ │
│ │ 公告栏：                                  │ │
│ │ ┌────────────────────────────────────┐ │ │
│ │ │ 公告内容/活动通知...                │ │ │
│ │ └────────────────────────────────────┘ │ │
│ │                                          │ │
│ │ 数据趋势图表：                            │ │
│ │ ┌────────────────────────────────────┐ │ │
│ │ │ 折线图/柱状图（活跃度、做题量等）    │ │ │
│ │ └────────────────────────────────────┘ │ │
│ └──────────────────────────────────────────┘ │
└──────────────────────────────────────────────┘
```

## 功能说明
- 顶部导航：Logo、系统名称、用户信息、退出
- 侧边栏：主要功能菜单
- 统计卡片：用户数、题库数、做题量等
- 快捷入口：常用操作按钮
- 公告栏：系统公告、活动通知
- 数据趋势：活跃度、做题量等可视化图表
